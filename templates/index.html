HOW DO WE DO IT?

Here's how you can operate our game.

Here are the instructions to operate our game. You can use the hand gestures given below and observe the changes and solve some calculations while having fun.

Left Hand

This gesture is used to decrease the value of the answer by 1. The minimum value we can go upto is -81.

Both Palms

This gesture is used to check whether the answer is correct or not.

Right Hand

This gesture is used to increase the value of the answer by 1. The maximum value we can go upto is 81.
